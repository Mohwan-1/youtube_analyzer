YouTube API 연결하기 👨‍💻

초등학생도 5분이면 OK! 단계별로 쉽게 따라해보세요

STEP 1

Google Cloud Console 접속

console.cloud.google.com에 접속하여 Google 계정으로 로그인하세요

💡 YouTube 채널과 같은 계정을 사용하세요!
STEP 2

새 프로젝트 생성

"새 프로젝트" 버튼을 클릭하고 프로젝트 이름을 입력하세요

💡 "YouTube-Analytics" 같은 이름을 추천해요
STEP 3

YouTube Data API v3 활성화

API 및 서비스 → 라이브러리에서 "YouTube Data API v3"를 검색하고 활성화하세요

💡 활성화까지 1-2분 정도 걸릴 수 있어요
STEP 4

API 키 발급

사용자 인증 정보 → "사용자 인증 정보 만들기" → "API 키"를 선택하세요

💡 생성된 키는 안전한 곳에 보관하세요
STEP 5

우리 서비스에 API 키 입력

아래 버튼을 클릭하여 발급받은 API 키를 입력하세요
